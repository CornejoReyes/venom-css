Grid System



The grid

In venom grid system, the .row represent a table row. This means that many columns can be part of it.

The colums has been defined automatically, so you don't need to specify a div as a column, but, you need to specify its size. This reduces the code you have to write.

<div class="row"> <div class="med-3"> <div class="med-3"> <div class="med-3"> <div class="med-3"> <div class="med-4"> <div class="med-4"> <div class="med-4"> <div class="med-6"> <div class="med-6"> <div class="med-12"> </div>

This means the unique column that row has will have the maximum size for a row: 12. A row can have columns from size 1 (smallest) to 12 (biggest).

Grid sizing

The columns need to complete the size of a row. Simply sume the size of the columns and try to complete twelve.

But, what means "med" exactly? This is the size of the sceen where code is displayed. If you are on a medium size screen, .med-* class helps you to respect the size of the column. If you use a small screen, .med-* class will ocupate 100% of the parent's width, but if you're on a large screen, .med-* will respect the size.

This is the responsive design. You can set multiple sizes to a HTML tag and, depends to the screen size it will resize itself.

Use .mob-*for extra small devices (moviles), .sml-* for small devices, .med-* for medium devices and .lrg-* for large ones.

Stylus variables

All the variables for grid are located at variables/grid.styl in the source folder. If you want to alter the number of max columns per row, only change $number-of-columns and that's it.



Row overflow

If you overflow the row, columns will start a new row.

Forms


Form basics

Most of the websites nowdays use forms, for sign-up or sign-in, make surveys and so on. Venom lets you to create pretty forms very easily.

First of all use the <form> tag to create a form. Add .form class and then add the <input> and <label> tags to build it.

Labels are the name of the input, so you can grup the label and input with .group class.

Add .label class to a <label> tag for properly apply styles. Use class .field on <input>

<form class="form"> <div class="group"> <label class="med-4 label">Name</label> <div class="med-8"> <input type="text" class="field"> </div> </div> <div class="group"> <label class="med-4 label">Last Name</label> <div class="med-8"> <input type="text" class="field"> </div> </div> <div class="group"> <label class="med-4 label">E - Mail</label> <div class="med-8"> <input type="email" class="field"> </div> </div> <div class="group"> <label class="med-4 label">Password</label> <div class="med-8"> <input type="password" class="field"> </div> </div> <div class="group"> <label class="med-4 label">Text Area</label> <div class="med-8"> <textarea name="" id="" cols="30" rows="10" class="field"></textarea> </div> </div> <div class="group"> <div class="med-left-4 med-3"> <button class="button primary full">Submit</button> </div> </div> </form>

Vertical form

By default, Venom makes horizontal forms, but you can make a vertical one. Just add .vertical to the element with .form class.

.form not requires to be in form tag. So you can simulate a form, but it won't work.

<input> tag with .field will ocupate the full width of its parent, so you can define a size with columns. In vertical form, <label> no need a column class, but they must have .label.

<form class="form vertical"> <div class="group"> <label class="label">Name</label> <div class="med-8"> <input type="text" class="field"> </div> </div> <div class="group"> <label class="label">Last Name</label> <div class="med-8"> <input type="text" class="field"> </div> </div> <div class="group"> <label class="label">E - Mail</label> <div class="med-8"> <input type="email" class="field"> </div> </div> <div class="group"> <label class="label">Password</label> <div class="med-8"> <input type="password" class="field"> </div> </div> <div class="group"> <label class="label">Text Area</label> <div class="med-8"> <textarea name="" id="" cols="30" rows="10" class="field"></textarea> </div> </div> <div class="group"> <div class="med-3"> <button class="button primary full">Submit</button> </div> </div> </form>
